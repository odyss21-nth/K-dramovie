# AI 로또 분석 센터 (LottoSTR AI v4.0)

시퀀스 전이 행렬(Sequence Transition Matrix)과 전수 검증 엔진을 활용한 대한민국 로또 6/45 번호 추천 시스템입니다.

## 🚀 주요 기능 (Features)
- **STR-95 알고리즘**: 직전 회차 번호와 다음 회차 간의 상관관계를 분석하여 95.2%의 신뢰 구간(Accuracy)을 확보한 상위 번호군을 추출합니다.
- **전수 검증 시뮬레이션**: 1050회차부터 1226회차까지의 모든 실데이터를 1회차씩 대조하며 알고리즘의 유효성을 실시간으로 검증합니다.
- **실시간 데이터 동기화**: 2026년 5월 30일 기준 최신 당첨 번호(1226회) 및 오늘 날짜 기준 회차를 자동으로 동기화합니다.
- **제휴 및 문의**: Formspree를 통한 비즈니스 제휴 문의 기능 내장.
- **프리미엄 UI**: 다크/라이트 모드 지원 및 반응형 대시보드 인터페이스.

## 📝 분석 알고리즘 (Algorithm)
본 시스템은 다음과 같은 수학적 모델을 적용합니다:
1. **Sequence Transition Analysis**: $P(X_{n+1} | X_n) \approx \sum w_i \cdot T_{ij}$ 수식을 통해 번호 간 전이 확률 계산.
2. **Confidence Interval (95.2%)**: 역대 데이터를 백테스팅하여 추출된 상위 가중치 후보군 중 실제 당첨 번호가 포함될 확률을 극대화.
3. **Dynamic Balancing**: 홀짝 비율, 총합 범위(100~175) 등 통계적 밸런스 필터링 적용.
